✅ ПРОЕКТ СОЗДАН И ГОТОВ К ИСПОЛЬЗОВАНИЮ!

📂 Структура проекта:
   ├── index.html          - Главный файл
   ├── css/style.css       - Стили
   ├── js/player.js        - JavaScript логика
   ├── audio/              - Папка для ваших MP3 файлов
   ├── images/             - Обложки альбомов (SVG заглушки)
   └── README.md           - Подробная документация

════════════════════════════════════════════════════════════
🚀 КАК ЗАПУСТИТЬ:
════════════════════════════════════════════════════════════

1. Откройте файл index.html в любом современном браузере
   (Chrome, Firefox, Edge, Safari, Opera)

2. Плеер запустится с демо-треками и заглушками обложек

3. Для добавления своей музыки - см. инструкции ниже ↓

════════════════════════════════════════════════════════════
🎼 КАК ДОБАВИТЬ СВОЮ МУЗЫКУ:
════════════════════════════════════════════════════════════

Шаг 1: Добавьте MP3 файлы в папку audio/

Шаг 2: Добавьте изображения обложек в папку images/

Шаг 3: Откройте js/player.js в текстовом редакторе

Шаг 4: Найдите метод initTracks() (примерно строка 67)

Шаг 5: Измените массив trackList:

  this.trackList = [
    {
      title: 'Мой любимый трек',
      artist: 'Любимый исполнитель',
      path: 'audio/моя-песня.mp3',
      cover: 'images/моя-обложка.jpg',
      duration: 180  // длительность в секундах
    },
    // добавьте еще треки...
  ];

Шаг 6: Сохраните файл и перезагрузите страницу!

⌨️ ГОРЯЧИЕ КЛАВИШИ:
════════════════════════════════════════════════════════════

  Пробел  →  Воспроизведение/Пауза
  →       →  Следующий трек
  ←       →  Предыдущий трек
  ↑       →  Громкость +5%
  ↓       →  Громкость -5%
  M       →  Выкл/Вкл звук
  R       →  Режим повтора
  S       →  Случайное воспроизведение
